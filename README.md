# Mourinha — Personal Portfolio

Personal portfolio website built from scratch with **HTML**, **CSS** and **vanilla JavaScript**.

## 🔗 Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — hero, about, skills, projects, socials |
| `interests.html` | Areas of interest: Cybersecurity, AI, Data Engineering, Design |
| `teachers.html` | Inspiring professors at Cesar School |

## ✅ Project Requirements (Checklist)

- [x] **Layout** — Flexbox and CSS Grid used throughout all sections
- [x] **Semantic tags** — `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`
- [x] **Multiple pages** — index, interests and teachers (3 pages total)
- [x] **JavaScript feature** — typing animation, IntersectionObserver reveal, bilingual i18n (EN/PT), dark/light theme toggle, particle canvas background, music player, scroll-aware navbar
- [x] **Valid markup** — validated via [W3C Markup Validation Service](https://validator.w3.org)
- [x] **README** — this file, with multiple commits tracking content updates

## 🛠️ Tech Stack

- HTML5 (semantic)
- CSS3 (Flexbox, Grid, custom properties, animations)
- Vanilla JavaScript (ES6+)
- [Spline](https://spline.design) — 3D scene in the hero section

## ✨ Features

- **Bilingual (EN/PT)** — full i18n toggle in the header, re-types the hero title on switch
- **Dark / Light theme** — system-wide toggle with smooth transitions
- **Typing animation** — hero title types character by character
- **Scroll reveal** — sections fade and slide in with IntersectionObserver
- **Particle background** — subtle animated canvas particles
- **Music player** — optional background music toggle (RHCP 🎸)
- **CV download** — English and Portuguese résumé available in the footer
- **Responsive** — works on mobile, tablet and desktop
